Welcome to keats

keats accepts images and guesses their colourmap. Use the /api endpoint.

For example, you can send an image of some seismic, and it will guess the colourmap and thius recover the data from the image.

Here's an example, using this image:

http://keats.geosci.ai/api?url=http://www.signatureseismic.com/user_files/images/fault.jpg

Parameters

  • url — the URL of the image to analyse.
  • region   the method for averaging:
    • auto — guess the data area.
    • off — do not guess the data area or crop.
    • the region to analyse in pixels, like 100,100,900,900
  • recover — whether to recover the data or not.
  • n_colours — how many colours to cluster (default 128)
  • return_cmap — whether to get the discovered colourmap back.
  • format — number of traces to extract to get statistics from (default 10)
    • png — a greyscale PNG image.
    • npy — a binary NumPy array.
    • txt — a text NumPy array.

Example

This GET request is the same as the one shown in the link above.

import requests

url = "http://keats.geosci.ai/api"
target = "http://www.signatureseismic.com/user_files/images/fault.jpg"
params = {'n_colours': 100,
          'url': target
          }
headers={"Accept": "application/json"}

response = requests.get(url, headers=headers, params=params)
response.json()
